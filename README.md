Fiji macro's that can be used for processing (confocal) images with multiple channels.
Split_edit_save_merge_flatten_save works as follows: splits images into individual channels, asks for values for each channels, edits the channels, merges them, flattens the images and saves the result.
Quantify_internalization selects single cells in one slice of a stack and then measures the integrated density of the particles within and writes the results to excel.
random_montage is usefull when you want to check the results of a large group of edited images. It randomly pulls images from a folder and makes a montage of them.
